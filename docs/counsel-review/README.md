# Counsel Review Packet

This folder is the working package for legal review. It is meant to describe
the platform's actual operating model before final legal copy is approved.

Use this packet to give counsel:

1. A short description of what FlatOrigin does and does not do.
2. The current moderation, reporting, and copyright workflows.
3. The current privacy, retention, and account-lifecycle behavior.
4. The current contractor document-review workflow and badge logic.
5. A list of issues that still need legal decisions.

Suggested review order:

1. `platform-positioning.md`
2. `moderation-and-dmca.md`
3. `privacy-and-data.md`
4. `verification-sop.md`
5. `open-questions.md`

Current product routes relevant to this review:

- `/terms`
- `/privacy`
- `/copyright`
- `/profile/edit`
- `/profiles/:username`
- `/messages`
- `/dashboard`
- `/admin/`

Internal note:

This packet is operational guidance and product context. It is not legal advice
and should not be published to users as final legal language.
